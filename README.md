# Patient Management System (C++)

A console-based patient management system for a clinic that assigns
priority scores to patients based on urgency.

## Features
- Add new patient records
- Assign priority score
- View patient list based on priority
- Basic data handling using C++

## Tech Stack
- Language: C++
- IDE: Dev-C++
- Concepts Used:
  - Functions
  - Arrays / Structures
  - Conditional logic
  - Priority-based decision making
